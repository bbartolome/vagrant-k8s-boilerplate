# vagrant-k8s-boilerplate

## Getting Started

**NOTE:** Requires [Vagrant] (https://www.vagrantup.com/) and [VirtualBox] (https://virtualbox.org) to be installed.

```
vagrant up
vagrant ssh
```

## Install

```
sudo ./setup.sh
sudo shutdown -r now
```
